defaultdict を使うと便利

デフォルト値を指定できる。

from collections import defaultdict

dct = defaultdict(int)

for x in arr:
    dct[x] += 1

list の場合

dct = defaultdict(list)
for x,y in arr:
    dct[x].append(y)

初期値を変えたい場合

dct = defaultdict(lambda: 10)

リストの代わりに dict を使う

3重配列などを宣言するのは面倒なため、 特に defaultdict と組み合わせると実装が楽になる。

Python では tuple をキーに取れることを利用して以下のように書ける。

from collections import defaultdict

dp = defaultdict(int)
dp[0,0,0] = 1

90度回転

def rot(s):
    '''
    二次元配列を90度回転
    '''
    return list(zip(*s[::-1])

座標圧縮

def compress(A):
    '''
    座標圧縮
    compress([1, 2, 3, 123, 1, 3, 4, 61231, 1, 3]))
     → [0, 1, 2, 4, 0, 2, 3, 5, 0, 2]
    '''
    xmap = {x: i for i, x in enumerate(sorted(set(A)))}
    return list(map(lambda x: xmap[x], A))
